Detailed Description

Classe para cálculo da média ponderada de uma imagem.

Classe responsável por calcular a média ponderada dos canais de uma imagem colorida.

Esta classe fornece métodos para calcular a média ponderada dos canais de cor de uma imagem utilizando pesos específicos para cada canal (B, G, R).

Esta classe fornece métodos para converter uma imagem colorida em escala de cinza, utilizando pesos personalizados para cada canal (B, G, R).

Calcula a média ponderada dos canais de uma imagem colorida.

Este método recebe uma imagem e os pesos para os canais azul, verde e vermelho, retornando uma nova imagem resultante da média ponderada.
