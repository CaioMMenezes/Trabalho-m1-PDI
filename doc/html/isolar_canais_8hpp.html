Detailed Description

Funções para isolar canais de cor em imagens coloridas usando OpenCV.

Este arquivo contém declarações de funções para isolar os canais vermelho (R), verde (G) e azul (B) de uma imagem colorida representada por um objeto cv::Mat.

cv::Mat isolarCanalB ( const cv::Mat & imagem)

Isola o canal azul (B) de uma imagem colorida.

Esta função recebe uma imagem colorida no formato BGR e retorna uma nova imagem onde apenas o canal azul é mantido, enquanto os canais verde e vermelho são zerados.
