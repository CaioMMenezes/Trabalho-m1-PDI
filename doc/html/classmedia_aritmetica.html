Detailed Description

Classe responsável por aplicar a média aritmética em imagens.

Classe responsável por calcular a média aritmética dos canais de uma imagem colorida.

Esta classe fornece métodos para calcular a média aritmética dos pixels de uma imagem utilizando a biblioteca OpenCV.

Esta classe fornece métodos para converter uma imagem colorida em uma imagem em escala de cinza, utilizando a média aritmética dos valores dos canais de cor.

cv::Mat mediaAritmetica::calcularMedia ( const cv::Mat & imagem)
static

Calcula a média aritmética dos pixels de uma imagem.

Calcula a média aritmética dos canais de uma imagem colorida.

Este método recebe uma imagem do tipo cv::Mat e retorna uma nova imagem resultante da aplicação da média aritmética.

Parameters
imagemImagem de entrada do tipo cv::Mat.
Returns
cv::Mat Imagem resultante após a aplicação da média aritmética.

Este método percorre cada pixel da imagem de entrada, calcula a média dos valores dos canais B, G e R, e gera uma nova imagem em escala de cinza.

Parameters
imagemImagem de entrada no formato cv::Mat (espera-se uma imagem colorida).
Returns
cv::Mat Imagem resultante em escala de cinza. Retorna uma matriz vazia se a imagem de entrada estiver vazia.
